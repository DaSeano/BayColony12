BayColony12

The Creation of Dune, a Sol Gov & Nanotrasen partnered Colony on a desert planet. This repo utilises, and will keep in sync with, BayStation12's code base.

View project onGitHub

BayColony Edits

All further BayColony Commits will begin with #BC and then an ascending numeric identifier. All PRs will undergo discussion. Major building maps must be at least publicly posted in the mapping thread for discussion.

For now, approval of content for the first iteration of the map will rest solely with Vetinarix. This is to ensure an accelerated development process that isn't bogged down in public indecision. At least not until later, anyway. When the first iteration of the map is complete, BayColony will trend towards a public approach to decision making.

BayColony Branches:

  • Master -- will be where BayColony12 Resides
  • Dev -- will keep in sync with BS12's dev branch.
  • Dev-Freeze -- will be for testing merges between BS12 dev branch code and BayColony12 master code.

Spriters

Within reason, you'll have a lot of leeway with these tasks:

  • Planetary Fauna (Think lizards and snakes, but also herd animals for hunting, and maybe some lion/gazelle/zebra equivalents. Walking cacti?)
  • Planetary Flora (Don't just give us cactus. Give us cotton monster plants and trees which grow upside down!)
  • Uniforms for Dune Colony Police Department
  • Uniforms for (evil) Nanotrasen Scientists
  • Casual Clothes (go nuts)
  • Fixing small items that may have NT branding, or don't match the look/feel (NT sec armor, PDA colours, ID cards, etc)

Coders

The following is a (slightly ambiguous) rundown of the major tasks which will require coders! Cookies will be presented to amazing people! Consult Vetinarix for detailed requirements and implementation discussion.

  • Night/Day cycle (Affecting solar power and default lighting state. The current magic number is a cycle lasting 1.5 hours, but it's up for debate.
  • Planetary Atmosphere & Game Areas (Not yet set in stone but: about equal to earth pressure, probably nitrogen based - must still contain a bit of oxygen, whatever its formation is, 60 degrees Celsius, accelerates hunger in humans, and slowly does burn damage across any non-covered parts of the body (hands, head). A new area tree will needed to be drafted for the locations around the colony.
  • Optimization of any atmosphere code for a large planet area.
  • Job System (Round Start/Joinable jobs shrink to the 6 Administration positions and their 5 deputies. Only other available joinable "job" is Colonist, which has no access by default. ID Computer must be able custom allocate door access codes (i.e add number 101 to the ID's access).
  • Planetary Fauna (Once we've got sprites. Feel free to sprite and code them if you really want your half-ape half-snake oasis monster and don't think anyone else is willing to help you with it)
  • Planetary Flora (Once we've got sprites!)
  • New terrain and terrain structures (oases, rocks, etc)
  • Possible changes to finance/economy (wages?)
  • Arrivals Shuttles, Cargo Shuttle & Cargo Vessel.
